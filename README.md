# PraisonAI examples

This repository mirrors the [`examples`](https://github.com/MervinPraison/PraisonAI/tree/main/examples) directory from [PraisonAI](https://github.com/MervinPraison/PraisonAI).

## Sync

GitHub Actions runs on a schedule (and can be run manually) to pull the latest `examples/` tree from the main PraisonAI repository and open a commit when there are changes.

## Documentation

See the [PraisonAI documentation](https://docs.praison.ai) for installation and guides.
